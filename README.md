# Zona Gym Pro

Aplicación web para registrar datos del alumno y generar un plan más completo de nutrición + entrenamiento.

## Incluye

- Fondo visual tipo gimnasio para una experiencia más atractiva.
- Cálculo de calorías de mantenimiento y objetivo (según meta).
- Porcentaje y gramos de macronutrientes con barras visuales.
- Proyección orientativa de cambio semanal y mensual de peso.
- Sugerencias de comidas orientadas a público peruano (costa, sierra, selva y provincias), priorizando opciones económicas de clase media-baja.
- Cada sugerencia indica porción y macros por plato (proteína, carbohidratos, grasas), además de etiqueta de costo y zona.
- Rutina sugerida de Lunes a Sábado (objetivo: bajar, mantener o subir peso).

## Cómo ejecutar

```bash
python3 -m http.server 8000
```

Abrir en navegador:

```text
http://localhost:8000
```

## Nota importante

Las recomendaciones son orientativas. Para un plan clínico personalizado, consultar nutricionista y entrenador.
